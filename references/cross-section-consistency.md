# Cross-Section Consistency — the "Golden Thread"

A whole-manuscript coherence pass (Mode C, step 5). Most "clinical feel" and reviewer trust comes not from any single section but from the thread that runs **Title → Abstract → Introduction aim → primary outcome → Methods → Results headline → Discussion main finding → Conclusion**. These checks are cross-cutting and **cannot be done section-by-section** — run them on the whole manuscript at once, after per-section polishing.

For each break you find, report its location(s), severity, and the minimal fix. Do not silently rewrite across sections.

## 1. Thread identity (aim → answer traceability)
Pin ONE primary research question and ONE primary outcome as fixed objects, then verify every section names the same thing:
- Title names the same endpoint/predictor/comparison + population as the Abstract.
- Abstract objective == Introduction aim (same verb, same marker, same endpoint, same population).
- The primary outcome defined in Methods is the one led with in Results and the one the Conclusion answers.
- The Conclusion answers the aim that was actually posed — not a different, easier question.

Flag: aim drift (Intro asks X, Conclusion answers Y); a secondary finding promoted to the headline; an endpoint reported in Results but never defined in Methods.

## 2. Numeric reconciliation
Every number appearing in more than one place must match exactly:
- Sample sizes, group n's, and the flow-diagram totals (Abstract == Methods == Results == Tables == flow diagram).
- The headline effect size, CI, and P reported identically in Abstract, Results text, and Tables.
- Event rates/percentages and their denominators consistent throughout.

Treat any mismatch as high-severity — reviewers read numeric drift as carelessness about the whole paper.

## 3. Terminology / abbreviation / label consistency
- One term per concept (decide sHT vs sICH; mTICI vs eTICI; define "good outcome" once as, e.g., mRS 0–2 and use it identically).
- Every abbreviation defined at first use, then used consistently (not re-defined, not alternated with the long form).
- Outcome labels, group names, and scale cut-offs identical across Methods/Results/Tables/Discussion.

## 4. Claim gradient (strength-of-claim coherence)
- Certainty escalates consistently: Results state what was found; Discussion interprets; the Conclusion must not exceed what Results support.
- No causal language for an observational/associational design anywhere in the chain.
- Hedging is consistent — the Conclusion should not be more confident than the Abstract or the Results.
- At most one restrained "first study" claim across the whole paper.

## 5. Methods ↔ Results ↔ Discussion fidelity
- Everything prespecified/described in Methods has a corresponding Result; every Result traces back to a Method.
- Subgroup / sensitivity analyses discussed were actually specified and run.
- The Discussion introduces no new results.

## 6. Journal-box ↔ body consistency
- Structured boxes (Eur Radiol Key Points / Clinical relevance; JAHA What is new / Clinical implications; the structured abstract) must restate the body — never add or contradict it.
