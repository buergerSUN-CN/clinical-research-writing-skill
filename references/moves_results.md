# Results moves

Report what happened before explaining what it means.

## Start with participant flow {#results-flow}

- Decision: give screened, excluded, enrolled/randomized, analyzed, and followed-up counts before the
  headline outcome when the source provides them.
- Evidence: invariant across the 32-paper families.
- Failure mode: starting with the most favorable estimate while the denominator remains unclear.

## Keep denominators recoverable {#results-denominators}

- Decision: report count before percentage and keep the relevant N visible, especially after missing
  data or subgroup restriction.
- Evidence: high-frequency corpus behavior.
- Failure mode: percentages whose denominator silently changes between sentences.

## Bind estimates to uncertainty {#results-estimates}

- Decision: name the effect measure and attach confidence interval and exact P value when reported;
  retain reference group and adjustment status.
- Evidence: strongest in observational models and randomized comparisons.
- Failure mode: `significant` without estimate, uncertainty, or direction.

## Order by the analysis plan {#results-order}

- Decision: cohort description → primary endpoint/comparison → adjusted or diagnostic analysis →
  secondary outcomes → safety → sensitivity/subgroup analyses.
- Evidence: recurrent across families.
- Failure mode: significance-first ordering that obscures the prespecified hierarchy.

## Use family-specific currency {#results-currency}

- Observational/prediction: event counts, adjusted associations, discrimination/calibration, and
  validation as applicable.
- Diagnostic/imaging: sensitivity/specificity, AUC, agreement, reader results, or measurement contrasts.
- Procedural: technical success, workflow, clinical outcomes, and complications kept distinct.
- Trial: arm-level outcomes, effect measure, uncertainty, analysis population, and harms.

## State null findings flatly {#results-null}

- Decision: report the estimate and uncertainty without rescue language.
- Evidence: real papers did not routinely editorialize nonsignificance.
- Failure mode: `a promising trend` or mechanistic explanation inserted into Results.

## Keep interpretation out {#results-no-interpretation}

- Decision: reserve plausibility, implications, comparison with literature, and recommendations for
  Discussion.
- Failure mode: Results paragraphs ending with `suggesting that clinicians should...`.
